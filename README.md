# Домашнее задание к занятию "`Репликация и масштабирование. Часть 2`" - `Герасимов Антон`

### Задание 1

Опишите основные преимущества использования масштабирования методами:
* активный master-сервер и пассивный репликационный slave-сервер;
* master-сервер и несколько slave-серверов;

*Дайте ответ в свободной форме.*

#### Ответ к заданию 1

Активный master-сервер и пассивный репликационный slave-сервер. В этой конфигурации Master обрабатывает все запросы (и чтение, и запись), а Slave только синхронизируется с Master, но не отвечает на клиентские запросы в обычном режиме.

Преимущества данного метода:
* Отказоустойчивость
* Резервное копирование без остановки
* Аварийное восстановление
* Защита от ошибок

Master-сервер и несколько slave-серверов. В этой конфигурации Master обрабатывает все операции записи (INSERT, UPDATE, DELETE), а несколько Slave-серверов распределяют между собой нагрузку по чтению (SELECT).

Преимущества данного метода:
* Распределение нагрузки чтения
* Высокая производительность
* Выделенные серверы для разных задач
* Обновления без простоя
